Flashcard

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que mostra cards com perguntas e assim que a pessoa que estiver no site clicar sobre a pergunta, haverá uma alteração sobre a pergunta com a respectiva resposta.

O HTML estrutura o conteúdo da página, dividindo-o em seções como a tela inicial, caixa de perguntas, resposta, a rotação dos componentes para que resultado final apareça. O CSS é utilizado para estilizar a página, empregando um esquema de cores futurista e layout responsivo para melhorar a experiência do usuário. JavaScript é usado para adicionar interatividade ao jogo, manipulando a rotação dos cards para atualizar o conteúdo da página conforme as perguntas, e determinando as respostas.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

AluraBooks

No Scratch, é possível criar um jogo interativo de ping pong simples e divertido. Nesse projeto, o jogador controla uma raquete usando as teclas do teclado(setas), para impedir que a bola passe para o seu lado, rebatendo-a contra o adversário que respectivamente é um robô. O jogo simula a dinâmica real do Ping Pong, com a bola se movendo de forma rápida e respondendo ao impacto das raquetes.

O código do jogo envolve o uso de blocos para detectar colisões, controlar a direção e velocidade da bola, além de pontuar cada vez que a bola ultrapassa uma raquete. O Scratch torna a criação desse tipo de jogo acessível e personalizável, permitindo que os jogadores ajustem aspectos como o fundo da tela, a cor das raquetes e o comportamento da bola, criando uma experiência interativa única para quem joga ou desenvolve o projeto.

A importância das ferramentas utilizadas, como os blocos de código visual do Scratch, reside na sua acessibilidade e na capacidade de engajar aprendizes de diferentes idades no pensamento computacional e na exploração de conceitos de forma interativa. O uso de cenários e movimentos ajuda a criar uma experiência de aprendizado mais rica e memorável, encorajando os usuários a explorar e interagir com o conteúdo de maneira ativa, em vez de passivamente.
